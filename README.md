# 🎬 Netflix Clone

A pixel-perfect frontend clone of the Netflix landing page. This project demonstrates proficiency in HTML5, CSS3, and Vanilla JavaScript, featuring dynamic content rendering, 3D card animations, and interactive UI elements.

## ✨ Features

- **Dynamic Content Rendering:** Movies and TV shows are loaded dynamically from JavaScript arrays, making it easy to add or remove content.
- **Category Filtering:** Toggle between "Movies" and "TV Shows" in the Trending section with a single click.
- **3D Flip Cards:** Hover over any trending movie/show card to see a smooth 3D flip animation revealing its title and description.
- **Interactive Carousel:** The trending carousel shifts left or right based on your mouse hover position for a premium feel.
- **FAQ Accordion:** Clickable FAQ section that expands and collapses answers dynamically.
- **Responsive Design:** Fully responsive layout that adapts seamlessly to mobile, tablet, and desktop screens.
- **Netflix UI/UX:** Dark theme with the signature Netflix red (`#e50914`) accents and modern typography.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure and layout.
- **CSS3:** Flexbox, CSS Grid, 3D Transforms (`transform-style: preserve-3d`), and hover animations.
- **JavaScript (ES6):** DOM manipulation, event listeners, and dynamic data rendering.

## 🚀 How to Run

1. Clone or download this repository to your local machine.
2. Locate the `index.html` file in the project folder.
3. Double-click `index.html` to open it in your default web browser.
